# Guitar-SolidWorks-Model
A 3D CAD model of an acoustic guitar designed using SolidWorks. This project demonstrates parametric modeling, part design while following standard CAD design practices.

 **Project Overview**
 
The objective of this project is to create a realistic 3D model of an acoustic guitar using SolidWorks.

 **Software Used**:
SolidWorks 2021

**Components**

• Guitar Body
• Neck
• Headstock
• Bridge
• Strings
• Sound Hole
• Frets
